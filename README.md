# Caddy Web Server with Docker Compose

Easy to use Caddy Web Server for Docker just 3 simple steps

## Getting started

1. Edit Caddyfile change `your.domain.com` to your domain

2. Run
```
docker-compose build
```

3. Run
```
docker-compose up
```
